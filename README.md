# 🍽️ Zomato Clone

A full-stack food delivery web application inspired by Zomato. Users can explore restaurants, browse menus, and place orders with a smooth and responsive experience.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)  
- 🔍 Search & Filter Restaurants  
- 📋 Dynamic Menu Display  
- 🛒 Order Placement System  
- 📱 Fully Responsive Design  

---

## 🛠️ Tech Stack

**Frontend:**  
- HTML, CSS, JavaScript / React  

**Backend:**  
- Node.js, Express  

**Database:**  
- MongoDB
